ලොව පුරා බිලියන දෙකකට අධික පරිශීලක සංඛ්‍යාවක් සහිත ජනප්‍රියතම chat platform එකක් වෙන WhatsApp, දැනටමත් Mobile App එකක් හා Web version එකක් ලෙස PC වලත් භාවිතා කළ හැකි බව ඔබ දැනටමත් දන්නවා ඇති.

කොහොම වුණත් මෙහි Web version හි User interface හා පහසුකම් ආදිය පිළිබඳ එතරම්ම සෑහීමකට පත්විය නොහැකි බව ඔබ දැනටමත් දන්නවා ඇති. මේ නිසා, desktop සඳහා සහය දක්වන පරිපූර්ණ WhatsApp PC App එකක් අවශ්‍යතාවය කාලයක් තිස්සේම බොහෝ දෙනාගේ අවධානය යොමු වී තිබුණු කාරණයක්.

මේ වන විටත් Windows සඳහා WhatsApp beta සංස්කරණයක් නිකුත් කර තිබෙන නමුත්, ජනප්‍රිය ඉතාලි වෙබ් අඩවියක් වන Aggiornamenti Lumia විසින් පළ කරන ලද screenshot හා GIF කිහිපයක් මූලාශ්‍ර කරගනිමින්, GSMArena වෙබ් අඩවි වාර්තා කරන්නේ නුදුරේදිම Windows හා MacOS සඳහා විශේෂයෙන්ම optimize කරන ලද පරිපූර්ණ අලුත් WhatsApp PC App එකක් එළිදැක්වීමට නියමිත බවයි.

මෙය Windows 11 හි theme එක වෙන transparent color panel සහිතව නිර්මාණය කරනු ඇති අතර, App එකෙන් ඉවත්ව සිටියදී පවා notification ලැබෙන ලෙස නිර්මාණය කෙරෙනු ඇති.

ඒ වගේම දැනට ඇති WhatsApp web version එකට log වීම සඳහා Process කීපයක තත්පර කීපයක්ම වැය කිරීමට සිදු වී තිබෙන නමුත්, මෙම නව PC App එක ඔබට තත්පරයකටත් වඩා අඩු කාලයකදී Open කර ක්‍රියාත්මක කරවීමට හැකියාව ලැබෙනු ඇති.

ඒ වගේම Touch display සඳහා සහාය දෙන පරිදි, drawing feature එකක්ද මෙම App එකට ඇතුළත් වෙනු ඇති බව ද වාර්තා වෙනවා.
